Tools Overview

Tools are software packages that perform specific tasks. We do not run tool directly in the DE, instead we create analyses for specific uses of installed tools. Only one tool or many tools may be used in an analysis. See Tool Integration and Creating a new Analysis in the Discovery Environment for more information.
