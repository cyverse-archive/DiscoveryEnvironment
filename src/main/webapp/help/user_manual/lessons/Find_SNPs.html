Find SNPs

An overview of Find SNPs is available.

Select Find SNPs from within Perform Analyses as described in that section. Click Ok.

Select input data

Zoom

Click Add to choose the previously uploaded SAM files in which you are seeking variants from the reference genome.

There is no limit to the number of files you may select here, but files must be selected one at a time. Delete will remove files previously selected during this step prior to launching the job.

Select Reference Genome

Zoom

Select the reference genome to which you will compare your SAM files.

Base Calling

Zoom

Select the base calling parameters.

The theta parameter, or error dependency coefficient, uses the maq consensus calling model and defines how much difference will be tolerated when calculating variance, assuming these differences to be natural fluctuations or other error rather than different sequences. For more details on these parameter settings, please see SAMtools and Maq.

Filtering

Enter your desired filtering parameters. For details on the filtering parameters, please see SAMtools.

Options

Zoom

Enter a name for your genotype sample to make it easier for you to keep track of multiple VCF data records.

The genotype name you input has different limitations than file and folder names elsewhere in the Discovery Environment. Here, you are limited to the characters described by this regular expression:

[A-Za-z0-9_#-.]{4,24}

Essentially, you may only use upper and lower case letters, numbers, and these symbols: _, #, . and -. The filename must be between 4 and 24 characters long and may not contain spaces.

Click Launch Job.

Enter a name and desciription for the job and click OK. See Perform Analyses for information about monitoring the process and where to find your results.
