Perform Analyses

Analyses take implemented tools and enable them to be executed in the Discovery Environment. Click the Perform Analyses icon on the main page to start.

Perform Analyses

Zoom

Perform Analyses is where you access tools that have been integrated into the Discovery Environment using specified interfaces. Click Choose Analysis to begin.

Choose Analysis

Analyses are categorized into logical groups to make specific tasks easier to find. Click the arrow next to a category to show what it contains. Select an analysis and click Ok to start.

When you have finished setting up your chosen analysis, click Launch Job.

Name and describe job

Enter a name for the job and write a description of it. Click Ok.

File name restrictions:
File names must be unique and may be a maximum of 250 characters. All alphanumeric characters are permitted, along with these special characters: the dash (-), underscore (_) or period (.). Spaces are allowed, but are not permitted as the first, last or only character.

Watch Analyses

Zoom

When you run an analysis, it will appear in Perform Analyses. The Status will update as the analysis is completed.

View Analyses

Zoom

After a completed run of an analysis, you can view the results. Select the analysis and then select View Output(s) from the drop-down menu at the right. You can also find View Outputs under More Actions.

To delete an analysis, select Delete from the drop-down menu at the right or from More Actions.

View Outputs (alternate)

Zoom

Analysis outputs are automatically placed in a folder in Manage Data and may be viewed from there at any time after a completed run.
